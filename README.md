**Alpha-Cars** is a modern car rental platform developed as part of a Bachelor of Computer Science project at the University of Tabriz. It leverages **React**, **Next.js**, **MongoDB**, **Prisma**, and **NextAuth** to provide a seamless, secure, and efficient rental experience.

## Installation & Run

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/alpha-cars.git
cd alpha-cars
```

### 2. Install Dependencies
```bash
# Using npm
npm install

# Using yarn
yarn

# Using pnpm
pnpm install
```

### 3. Run Development Server
```bash
# Using npm
npm run dev

# Using yarn
yarn dev

# Using pnpm
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app in action.
